# Luv Bugs Collection

The English-language site for **Luv Bugs Collection** — a children's
brand for ages 3–6 featuring Candy Bug, Love Bug, Happy Bug, Tease Bug,
Dance Bug and Rich Bug. Includes the *Feed Candy Bug* HTML5 game.

Static site — plain HTML, CSS and JavaScript. No build step.
Hosted on GitHub Pages at [luvbugscollection.com](https://luvbugscollection.com).

Swedish sister site: [bacillerna.se](https://bacillerna.se).
